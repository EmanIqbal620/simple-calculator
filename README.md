# Simple Calculator 🧮

> A CLI calculator supporting addition, subtraction, multiplication, division, exponentiation, and modulo  published on NPM.

[![NPM](https://img.shields.io/badge/NPM-CB3837?style=for-the-badge&logo=npm&logoColor=white)](https://www.npmjs.com/~eman_iqbal)

```bash
npm install simple-calculator
```

---

## Operations Supported

| Operation | Symbol | Example |
|---|---|---|
| Addition | `+` | 5 + 3 = 8 |
| Subtraction | `-` | 10 - 4 = 6 |
| Multiplication | `*` | 6 × 7 = 42 |
| Division | `/` | 20 ÷ 4 = 5 |
| Exponentiation | `**` | 2 ** 8 = 256 |
| Modulo | `%` | 10 % 3 = 1 |

---

## 🚀 Run Locally

```bash
git clone https://github.com/EmanIqbal620/simple-calculator
cd simple-calculator
npm install
npx ts-node index.ts
```
